NOID-mint Python Package

## Installation
```

git clone https://github.com/aprigge/NOID-mint.git
python setup.py install
```

## Usage
* Generate noid
```
format .yml file to include appropriate ark shoulder
noid -f [name of file].yml
```

## Testing
```
nosetests
```
